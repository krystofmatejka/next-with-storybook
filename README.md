# next.js template

Simple template for js web application

## Contains

 - next.js
 - react
 - typescript
 - styled components
 - jest
 - storybook

## Quick start

### Install dependencies

```bash
npm install
```

### Run dev server

```bash
npm run dev
```

### Run jest

```bash
npm test
```

### Run storybook

```bash
npm run storybook
```
